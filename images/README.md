## Image Guidelines

- **Format**: Use JPG for photos, PNG for graphics with transparency
- **Size**: Optimize images for web (compress to reduce file size)
- **Dimensions**: Follow recommended dimensions for consistent display
- **Quality**: Use high-quality images that represent diversity and inclusion
- **Copyright**: Ensure all images are properly licensed or original
